Embed on WordPress Using iFrames

  • You can use iFrames to embed media on WordPress sites
  • For the iFrame HTML element, you need to find the iFrame "embed code" of the media you wish to embed
  • StoryMaps and Google My Maps have iFrame codes available
    • Look for embed code, which starts with <iframe src="URL">
  • Some content, such as YouTube videos, can be embedded just using the video URL

Find iFrame Information on StoryMap

1. To get the iFrame code from your StoryMap, click the “Share” button on the upper right hand corner

 

2. The iFrame code can be found under the heading “Embed”

3. Copy the code in the box

 

4. Now we can turn to WordPress!

Embed Media onto a WordPress site

  • To embed media on WordPress, copy and paste the iFrame code from StoryMap and WordPress will automatically embed it.
  • WordPress can automatically turn URLs into embedded content
  • To embed a YouTube video, copy and paste the video’s URL into a WordPress page
