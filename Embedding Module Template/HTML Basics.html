What is HTML?

  • HTML is the basic language used by web browsers to display web pages
  • HTML can be used to make changes to the way a web page is displayed and how it behaves
  • HTML relies on iFrames and image tags to embed material
  • <img> tags are like hanging a painting on your wall, while iFrames are like hanging a television on your wall
    • The <img> tag will just show a static image, while iFrames allow more dynamic content, such as an interactive map

What are iFrames?

iFrames (or Inline Frames) are a nested browsing context, embedding one HTML page into an existing one.

The iFrame HTML element is often used to insert content from another source into a web page. 

Websites use the iFrame element for embedding.
