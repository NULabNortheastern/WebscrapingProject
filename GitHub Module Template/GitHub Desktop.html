GitHub Desktop is an application that connects the cloud repository to your local computer.

First, you need to install GitHub Desktop and then connect your local computer to the cloud repository. In order to do this, in the “File” menu, click “Clone a Repository” and then use your repository’s URL.

Follow these steps to make changes (editing, uploading, deleting, etc.) to GitHub:

  1. You MUST CLICK “Fetch Origin”/”Pull” every time you want to make any edits, even if it’s a small one. This way, your local version is up-to-date with the cloud version. Otherwise, there could be GitHub conflicts.
  2. Make your edits on your local files/folders, whether that’s editing a README file, adding a new PDF, &c.
  3. When you want to upload your new files/folders into the cloud repository:
    1. Make sure the edited files are selected in the browser view on the left (if you’re on a Mac, unclick anything with DS_Store)
    2. Write a brief summary of what you are changing 
    3. Click “commit to master”
    4. Click “Push,” and voila, it should be on GitHub! You can look for your changes in GitHub on your browser to be sure.
