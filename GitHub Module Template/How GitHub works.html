GitHub is a cloud storage repository that connects to your local computer.

In order to make changes to files on GitHub, you can either

  • do so online using the GitHub site, or
  • make the changes on your own computer and then commit & push those to the online repository using GitHub Desktop.

The latter is usually the easiest way to create new folders, add new materials, etc., while the former often better for editing README and other markdown files.

To edit GitHub, you must have a GitHub account and must be a contributor/collaborator.
