To make changes (editing, uploading, deleting, etc.) to files on GitHub directly from your browser:

  1. Be sure that you’re logged into your GitHub account.  
  2. From your repository, navigate into the appropriate folder you need to add materials to, or edit.  
  3. You can drag and drop PDF or markdown files directly from your computer into the file by clicking the “add file” dropdown menu in the upper right corner of the window and selecting “upload file.” 

However, most GitHub users prefer to use its desktop version for modifying code and uploading materials. (Some users even prefer the "command line" for interacting with GitHub repositories; this is out of scope for our tutorial, but does have some more powerful options!)
