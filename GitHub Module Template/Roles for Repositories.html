Different roles have different permissions to work with the materials in your repository. 

For personal repositories, there are two types of roles. 

  • Owner: the person the repository belongs to; each repository can have only one owner 
  • Collaborator: someone that the repository owner has granted access to so that they can make changes to the repository

When a repository is used for an organization, there are various roles with varying credentials. 

  • You can invite people to collaborate on organizational repositories with a broader set of permissions, including reading, writing, and admin.
  • You can also invite people to join the organizations themselves as owners (here, there can be more than one) or members.
