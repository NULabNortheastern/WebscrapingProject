What is Git?

Git is a distributed, open-source version control system (VCS) that enables you to store code, track revision history, merge code changes, and revert to earlier code versions when needed. This allows you to manage multiple contributions while avoiding conflict

What is GitHub?

GitHub is a cloud software development platform. It is commonly used as a shared place to manage collaboratively-developed software and data. GitHub is a popular social platform for software development communities.  Many people contribute to the same set of files and thus rely on good version control.
