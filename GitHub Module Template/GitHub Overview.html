GitHub is a code hosting platform for version control and collaboration, using the Git version control system. It lets you and others work together on projects from anywhere.

This tutorial teaches you Git & GitHub essentials like repositories, branches, commits, and pull requests. You'll create your own repository and learn GitHub's pull request workflow.

In this brief tutorial guide, you will:

  • Learn how Git & GitHub work
  • Review most used Git terminology
  • Learn various roles for repositories
  • Use GitHub online and on your Desktop
  • Create and use a repository
  • Make changes to a file and push them to GitHub as commits
  • Open and merge a pull request
