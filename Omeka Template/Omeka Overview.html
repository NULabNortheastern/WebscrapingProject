Welcome to the Digital Integration Teaching Initiative's (DITI) module on Omeka. This module is an introduction to using Omeka, an open-source web publishing platform for creating and sharing digital collections and media-rich online exhibits. This module teaches basic terminology, how to navigate the program, how to add items and metadata, and how to organize items.

Learning Goals

  • Learn how to navigate Omeka and add items and metadata
  • Learn different ways to organize items in Omeka

Learning Objectives

Students will be able to use Omeka to curate items. Students will be able to successfully navigate Omeka and add items, metadata, collections, tags, item types, simple pages, and exhibits.
