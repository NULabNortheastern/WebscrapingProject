Item types are the different categorizations you can assign to an item. They are user-defined types of objects, with associated metadata. Omeka.net comes with predefined item types with fields specific to the item type, for instance, Document includes fields for document text and original format.  They also include oral history, still image, person, email, lesson plan, website, etc. You can easily edit these item types or add your own.

  • You can edit item types by clicking on “edit,” under the Item Types tab.
  • There, you can edit the description of an item type and its elements.
  • Elements are the different attributes of an item type.

  

Editing Elements for Item Types

  • You can add existing elements from a list or create new ones
  • You can also remove the default elements for an item type (but make sure you think carefully before doing this!)

Adding New Item Types  

  • You can add completely new item types by clicking the “Add an Item Type” button
  • You can give the new item type a name and description, and add elements to it (either existing or new).
