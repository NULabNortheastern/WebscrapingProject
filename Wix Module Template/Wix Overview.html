Welcome to the Digital Integration Teaching Initiative's (DITI) module on Wix.

This module will cover creating and designing a website using Wix. It demonstrates how to sign up for an account with the Wix website building tool, create a website, choose templates, and customize your website based on audience, goals, project themes, media to be included, and more.

Learning Goals

  • Understand the layout and choices available through Wix
  • Better articulate your intended audience and design choices to appeal to that audience

Learning Objectives

  • Create a Wix website with different pages, like Homepage, About Me, Blog.
  • Use specific features, like the “Add media” function
