If you want to add media to your website, use the following steps:

  1. Go to "Add", then "Add Section to Page" and click "Video" (or Media Gallery if you want to add multiple items).
  2. Select the location where you want the video section to be added.
  3. Select the layout you like best.
  4. You can also drag and drop the layout of your preference on the website.

 

 

When you add the section, you can edit the title, subtitle, text, the URL/video file, and all other aspects using the "edit" button.
