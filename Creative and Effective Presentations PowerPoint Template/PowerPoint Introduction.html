Microsoft PowerPoint is a presentation program for Windows, Mac or mobile devices. It can create presentations from scratch or a template. Users can add text, images, art, and videos; use professional designs with PowerPoint Designer; and add transitions, animations, and motion. Presentations can be saved to OneDrive and shared to work with others.

PowerPoint is available with Northeastern's Microsoft 365 subscription. You can see installation instructions for Mac here and Windows here. 

Google Slides also have similar functions and can be used in place of PowerPoint. Google Slides can be used for free. Much of the best practices for PowerPoint presentations also applies to presentations made using Google Slides.
