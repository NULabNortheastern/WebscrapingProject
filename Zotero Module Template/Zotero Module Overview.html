Summary of Module

Welcome to the DITI's module on using Zotero for content management. This module will go over the basics of how to install and use Zotero. The module introduces Zotero's different functions and features, and discusses how it can be effective for organizing research materials.

If you have not downloaded Zotero, you can download it through this link.

Learning Objectives

  • Understand how to use citation management software to organize research materials

  • Understand and define Zotero's functions and features, including libraries, ISBN, tags, and more.

  • Input bibliographic information both manually and automatically into Zotero

  • Transfer a library into a Works Cited page
