Summary of Module

Welcome to the DITI's module on using NVivo for qualitative text analysis. This module will go over the basics of how to install NVivo and demonstrate how to use the software as a literature review and research tool. The module introduces the basic concepts of NVivo and shows how to import documents, and how to create nodes, basic analytical visualizations and queries.

If you have not downloaded NVivo, you can find instructions on how to get it through Northeastern University at this link.

Learning Objectives

  • Understand the purpose of qualitative coding and how NVivo achieves this purpose
  • Define specific NVivo terminology
  • Understand the mechanics of creating an NVivo project and the layout of the user interface
  • Learn how to import files, and how to create nodes, analysis cases, code documents, and annotations
  • Use nodes to create visualizations and perform analytical queries like word frequency and cross-tabulation
