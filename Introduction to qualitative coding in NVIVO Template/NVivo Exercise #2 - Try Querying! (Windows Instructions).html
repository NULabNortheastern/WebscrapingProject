Exercise 2: Try Querying - Word Frequency

On Windows systems, Query can be found in the Explore Tab

  1. Upload the following files into NVivo:

    • Bandara, Wasana, “Using NVivo as a Research Management Tool: A Case Narrative”
    • Gregorio, Silvana di, “Using NVivo for your Literature Review”
    • Roth, J.J., “Using NVivo to Organize Literature Reviews”
  2. To query multiple items, select the items you would like to query in the Selected Items tab. For the sake of this exercise, let's select the files by Silvana Gregorio and J.J. Roth. 
        • Note: Alternatively, you can Control+click on a file and select Query
  3.  Click Run Query
