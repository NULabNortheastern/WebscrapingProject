There are two ways to import files into NVivo - through the command bar and through control-click/command-click (or right click using a mouse)  in Mac versions.

Mac

1) Going through the command bar: select the Data tab in the command bar at the top of the program window. From there you can select the type of file you want to import by clicking on the icons in the tab.

 

2) Control Click / Right click: in the menu to the left, select Files. In the list to the right you can double-click the trackpad on your laptop in the bottom right (if you are on a desktop double-click the right mouse button), and you will see the Import option. When you hover over it with the cursor, you can select the type of file you want to import.
