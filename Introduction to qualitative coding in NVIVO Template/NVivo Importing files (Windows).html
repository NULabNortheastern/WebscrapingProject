There are two ways to import files into NVivo - through the command bar and through control-click (or, right click if using a mouse) both in Windows and Mac versions.

Windows

1) Going through the command bar: select the Import tab in the command bar at the top of the program window. Then select Files. As you can see, various types of files can be imported.

  

 

2) Control-click / Right click: in the menu to the left, select Files. In the list to the right you can click the right mouse button, and you will see the Import Items... option.
