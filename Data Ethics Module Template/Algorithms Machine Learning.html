"Machine learning" happens when an algorithm tells a computer to make decisions based on a set of patterns derived from data, instead of following specific pre-determined instructions. (That is, the machine is instructed to "learn" what to do, based on particular input specifications.) 

Supervised machine learning is where a computer is supplied with characterized and labeled data to “train” the computer to recognize patterns based on the dataset. Computer vision facial recognition is an example of supervised machine learning. Modern facial recognition software packages train themselves to recognize faces by being fed hundreds of thousands of face images.

Unsupervised machine learning is where a computer is given uncharacterized data and allowed to build its own inference structure without the guidance of a human “trainer.” A great example is AlphaZero--a computer created by Google and programmers who only gave it the rules of chess. The computer then trained itself by playing chess against itself millions of times. AlphaZero was then able to beat existing chess programs and chess masters. Now, people are trying to learn from AlphaZero.
