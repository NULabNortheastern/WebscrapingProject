Frequently Asked Questions in DITI Workshops: 

How do I change my text color in KnightLab StoryMap?

  • To change the text color in KnightLab StoryMap you’ll need to use the <span> tag in HTML.
  • Click the HTML button (</>) in the editor. Then add the tag; for example: <span style ="color:red"> YOUR TEXT HERE </span>
      • Note: you must open and close the tags appropriately for the code to work.
  • Be sure to wrap the <span> tag around the entire text you wish to be in a different color.
  • Experiment with different colors! Try it on your own by changing the color from “red” to “purple”. 
  • You can find color names for HTML here: https://www.w3schools.com/colors/colors_names.asp

 

Can I change the zoom level of my map?

  • The zoom level that your user will see is determined by the distance between your locations.
    • A StoryMap whose locations are all within the city of Boston will be more zoomed in than one which includes locations on different continents.
  • There is no way to override this, so keep it in mind when designing your StoryMap.

 

How do I add “alt” (alternative) text to my images for accessibility purposes (i.e., screen readers and other assistive technologies)? 

  • To make an image accessible for screen readers, you can add alt text with HTML code, using this template:
    • <img src="IMAGE_LINK" alt="IMAGE_DESCRIPTION"/>
    • Note: typically, the <img> tag is associated with a specific image. However, the software for StoryMap separates the slide image from the caption. Be aware of this as you make accessibility-related improvements to your StoryMap. 
  • Alternatively, you can do longer-form, more detailed image descriptions within the caption section of the image slide. This may be the best approach considering software limitations.

 

Can I change the Text Fonts for my StoryMap?

  • Yes! Go to the "Options" button on the top left corner and you will find "Fonts" menu. You can change your StoryMap's font here.

Can I add a full website into my StoryMap?

  • Yes! To do this, paste the URL of whichever page of the website you'd like people to see first into the "Media" URL box.
  • The slide you add the webpage to will now display that webpage, with the landing page immediately visible. 
  • Viewers of your slide should be able to interact with the website (i.e., click around and go to different parts of the website without leaving StoryMap).
