Welcome to the Digital Integration Teaching Initiative's (DITI) module on StoryMap. In this module, we will introduce you to the following:

  • StoryMap as a mode of conveying data and content
  • Best practices and questions to consider before creating a StoryMap
  • Steps for using Knight Lab’s StoryMap

What is a “story map”?

Telling a story using maps! 

 “You combine authoritative maps with narrative text, images, and multimedia content. They make it easy to harness the power of maps and geography to tell your story.”  - ESRI Story Maps Website

To get a sense of how story maps work, let's briefly consider the example project below. Navigate to the hyperlink and click "Start Exploring." Learning Objectives

By the end of this module, you should be able to:

  • Articulate particular choices you will make when telling a story using a map
  • Follow a step-by-step guide for creating, saving, and publishing maps using KnightLab’s StoryMap
  • Upload data into StoryMap, including location information, images, and text
  • Navigate StoryMap’s map markers and location-finding system
