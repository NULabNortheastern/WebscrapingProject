Instructions

Following are instructions for the process of making a StoryMap: creating a new project, making a title slide, adding slides (with media, images, and links), and customizing your background. Please note this StoryMap module focuses on the use of cartography/geographical map and not the image/Gigapixel StoryMap option.

Create a New Project

  • Go to https://storymap.knightlab.com/
  • Click “Make a StoryMap”
  • After creating an account or signing in, click “New Map” and name your project

Make a Title Slide

  • The title slide will display all the map markers
  • The "Text" box on the bottom-right is where you put the project description
  • Add media for the page in the “Media” box (image/video, caption, credit) on the bottom-left

 

Add Slides (Map Markers)

  • Click the “Add Slide” button in the left-hand column 
  • Add a location
    • StoryMap uses Google Maps to locate addresses
    • Type the address or coordinates into the search bar in the center of the page
    • If you can’t find the location, you can drag and drop the map marker to a specific location
  • Add the description and media (just as with the title slide). Since only one media file can be uploaded per slide, you can use the text box’s HTML to integrate an image or another media type.

Add Media to Slides

  • To add media, enter a URL to an image, a video or an audio file into the "URL to your media" box. Don't forget to add credit and a caption for your media.
    • This works for images from Wikimedia Commons and other places, YouTube and Soundcloud.
    • You can also upload files saved locally on your device.
    • Always use the "embed code" for the image or find a link directly to the image file (that is, you need to link to the image file itself, not to a webpage that has an image on it). To access the appropriate link in Wikimedia Commons, go to "More Details", then "Use this File on the Web", and then select "File URL".
  • Add an audio file using SoundCloud
    • Upload your audio file to SoundCloud
    • Copy the URL of your audio file and paste it into the "URL to your media" box.
  • To add an interactive Google Street View map within a slide, enter the URL in the same box.
    • On Google Street View, click "Share or embed image," and then click the "Embed a map" tab.
    • Copy the URL, making sure it in the format <iframe src="URL"></iframe>.
    • This should happen automatically when you copy the link, but double check that the format is correct because the map won't work without it.

Add Media and Images in Text

  • To add additional media to a slide, you can use basic HTML
  • Click the </> button (turn on HTML)
  • Get a URL of your image. Wikimedia Commons is a great source for images. Make sure to use the embed file code, and don’t forget to cite and check the permissions for any images you use! You can find the embed code under the "Use this file on the web" section as described above.
  • Use the HTML <img src="URL"/> tag to insert the image.
  • Check the “Preview” to see the image

Add a Link in a Caption or Credit

  • Use the HTML <a href="URL">text</a>
  • Following the example above, fill in the link you want to use between the quotation marks, and replace the word "text" with the text you want to be linked.

Customize StoryMap

  • The “Options” button in the top left allows you to change the design of your map. Options include Language, Fonts, Call to Action, Map Type and more.
  • “Background Options” in the bottom right of the text box allows you to customize the color of the background. You can also use images as the background for your slides.

Publish and Share

  • The "Publish Changes" button is in the top left side on the editing page.
  • From this page, choose the "Sharing" tab in the top right corner. You can use the link or the embed syntax provided on this tab to share your map. You can directly share on social media sites as well. You can add a description and feature image here before sharing your StoryMap.
